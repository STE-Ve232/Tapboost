# **App Name**: TapBoost

## Core Features:

- Leaderboard Display: Display a leaderboard of top earners, updated periodically.
- Tap to Earn: Allow users to tap a button to earn virtual currency. Taps increase a user's earnings.
- Referral Program: Enable users to refer friends and earn a bonus for each successful referral.

## Style Guidelines:

- Primary color: Light blue (#3498db) to maintain a clean and trustworthy aesthetic.
- Secondary color: Green (#2ecc71) for positive actions like referral bonuses.
- Accent: Yellow (#f1c40f) for bonuses and special rewards.
- Clean and modern sans-serif fonts for a user-friendly interface.
- Simple and clear icons to represent different actions and rewards.
- Card-based layout to organize information effectively.
- Subtle animations on button taps and reward notifications to provide feedback.
